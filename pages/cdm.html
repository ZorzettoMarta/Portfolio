"I Colori di Matteo" Site

Group project exam Visual Communication and Interface Design, 2021

Goal and challenge tools

Our goal was to redesign a website of a non-profit Association. To complete our task we had to develop the site using:

  1. HTML5
  2. CSS
  3. Bootstrap
  4. Javascript

Design process

Before developing the website, we followed these steps to make it more user-friendly and engaging.

  1. We analyzed the existing website.
  2. We did a benchmark to take inspiration to create the new website concept.
  3. We built our proto-personas based on the target.
  4. We made a new concept and a mood board.
  5. We decided the palette, the typefaces and the icons to use.
  6. We defined the information architecture and the user flow.
  7. We rapidly prototyped the different web pages of the site by hand.
  8. We made the mockup of the site for desktop, tablet and smartphone, using Figma.
  9. Finally, we developed the website.
